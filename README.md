# 🏢 לו״ז משרד — Office Schedule PWA

אפליקציה לסנכרון לו״ז הגעה למשרד בין שותפים.
השותף מעדכן את שעות ההגעה והעזיבה — ואתה מקבל עדכון בזמן אמת.

## ✨ פיצ׳רים

- **שני תפקידים**: שותף (מעדכן) וצופה (מקבל עדכונים)
- **סנכרון בזמן אמת**: SSE — כל שינוי מופיע מיד אצל שני הצדדים
- **PWA — התקנה על הטלפון**: מופיע כאפליקציה עצמאית עם אייקון
- **הזנה בלחיצות בלבד**: בחירת שעות ע״י לחיצה, ללא הקלדה
- **תצוגות**: יום / שבוע / חודש
- **התראות**: פעמון עם ספירת עדכונים שלא נקראו
- **ללא הגבלת תאריכים**: אפשר לעדכן כמה שרוצים קדימה

## 🚀 התקנה

### דרישות
- Node.js 18+
- npm

### שלבים

```bash
# 1. התקן את החבילות
npm install

# 2. הרץ את השרת
npm start
```

השרת ירוץ ב: `http://localhost:3500`

### גישה מהטלפון

1. ודא שהמחשב והטלפון על **אותה רשת WiFi**
2. מצא את כתובת ה-IP של המחשב:
   - Mac: `ifconfig | grep "inet " | grep -v 127.0.0.1`
   - Windows: `ipconfig` (חפש IPv4 Address)
3. בטלפון, פתח בדפדפן: `http://<IP>:3500`
4. **להתקנה כאפליקציה:**
   - **iPhone (Safari):** כפתור שיתוף → "Add to Home Screen"
   - **Android (Chrome):** תפריט ⋮ → "Add to Home Screen" / "Install App"

### פריסה לאינטרנט (אופציונלי)

אם רוצים גישה מבחוץ (לא רק ברשת מקומית), אפשר לפרוס ל:
- **Railway** — `railway up`
- **Render** — חבר repo ל-Render
- **VPS** — העלה לשרת עם PM2: `pm2 start server.js`

⚠️ **חשוב**: לפריסה עם HTTPS (נדרש ל-PWA install בפרודקשן).
